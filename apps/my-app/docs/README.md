# My App

## Quickstart

```bash
cd apps/my-app
pnpm dev
```

## Configuration

- **App Name:** My App
- **Domain:** localhost:3000
- **Company:** Your Company
- **Locale:** en-US

## Development

- Dev: `pnpm turbo run dev --filter=my-app`
- Build: `pnpm turbo run build --filter=my-app`
- Test: `pnpm turbo run test --filter=my-app`
